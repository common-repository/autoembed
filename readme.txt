== Description ==

Allows Wordpress users to easily embed videos from over 150 supported video sites
using one common embed tag.

== Installation ==

1. Copy autoembed.php into your Wordpress plugins folder, normally located
   in /wp-content/plugins/

2. Login to Wordpress Admin and activate the plugin

A video player can be embedded in a post using a tag of the following
form:

[embed:URL]

Eg. [embed:http://www.youtube.com/watch?v=z3U0udLH974]
